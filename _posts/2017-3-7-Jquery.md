속성값으로 선택하기
$('img[alt=이미지 설명]');

셀렉터집합 순서대로 실행하기
$("input[type='button']").each(function() {
  	$(this).attr('blevel')
});

속성값이 존재하는지 확인해볼때
$(this).is("[속성명]")

- jquery and 조건 셀렉터(여러개의 조건으로 선택하고자 할 때)
```javascript
$("input[type=''][name*='']");
```
